Cara Reset Router

Dari latar belakang masalah lupa password admisitrator modem, sampai pengen ganti modem dengan yang baru. munculah permasalah untuk mereset modem dan menyeting ulang si router, agar bisa konek lagi dengan internet.

Hard Reset

Hard reset biasanya dilakukan ketika anda benar-benar lupa login administrtor router dan hard reset adalah cara yang paling simple untuk mereset router. untuk melakukan hard reset, anda hanya perlu mencari lubang kecil atau tombol reset dibelakang router anda, tekan kurang lebih 8-10 detik hingga lampu indikator mati semua lalu hidup kembali.

Soft Reset

Untuk melakukan soft reset anda harus mengetahui password adminitrator router anda.

  • Pastikan komputer anda terhubung ke router
  • Buka IP Gateway router anda
  • Login adminitrator
  • Pilih menu "system" (hampir semua router penamaan menu sama)
  • Pilih "reset router"
  • Pilih "reset"
  • Tunggu kurang lebih 1 menit & router anda sudah kembali ke pengaturan defaultt
